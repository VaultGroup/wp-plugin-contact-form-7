A simple plugin for capturing Contact Form 7 (WPCF7) enquiries in VaultRE

== Description ==
This plugin allows you to automatically send your enquiries from Wordpress Contact Form 7 to your VaultRE CRM account.

== Installation ==
1. Activate the plugin through the \"Plugins\" menu in WordPress.
2. Enter your API settings and form settings in the \"Settings\" menu in WordPress.
